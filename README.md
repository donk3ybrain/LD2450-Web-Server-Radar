# LD2450-Web-Server-Radar
Example code for displaying LD2450 radar targets in a web browser

Arduino IDE 1.8.19
ESP32 v3.0.7 Board Files
ESP32-S3 with 5V Out
Tested on WeAct S3 Mini H4R2

Connect ESP32-S3 to HLK-LD2450:

G to G
5v to 5v
17 to TX
16 to RX
